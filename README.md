BO3 Workshop Sync Tool
======================

A lightweight tool for syncing Call of Duty: Black Ops III Steam Workshop mods and custom maps with the Resurge client.

------------------------------------------------------------
HOW TO USE
------------------------------------------------------------

1. Launch the application (ResurgeSyncTool.exe).

2. The tool will automatically attempt to detect:
   - Call of Duty Black Ops III game folder
   - Steam Workshop folder (311210)

3. If needed, manually select the correct folders using the Browse buttons.

4. Click the "SYNC WORKSHOP" button.

5. Wait for the process to complete.

6. Synced content will be placed into:
   - Call of Duty Black Ops III/usermaps
   - Call of Duty Black Ops III/mods

Each map/mod will be automatically structured with a "zone" folder containing all required files.

------------------------------------------------------------
FEATURES
------------------------------------------------------------

- Auto-detects BO3 and Steam Workshop directories
- Separates mods and maps automatically
- Builds correct zone folder structure
- One-click syncing

------------------------------------------------------------
REQUIREMENTS
------------------------------------------------------------

- Windows
- Call of Duty: Black Ops III installed via Steam
- Steam Workshop content downloaded (AppID: 311210)

------------------------------------------------------------
USAGE NOTE
------------------------------------------------------------

This tool is intended for personal use and for simplifying
mod/map transfer between Steam Workshop and the Resurge client.
